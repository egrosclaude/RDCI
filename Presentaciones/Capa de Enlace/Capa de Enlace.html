  • Capa de Enlace
    • Principios
      • Objetivos
        • Transportar datagramas entre nodos adyacentes
        • Con control de integridad
        • Con confiabilidad
          • Cuando sea preferible
            En casos de enlaces
            con alta tasa de error
            (ej. inalámbricos)
      • Contexto
        • Capa de Red
          • PDUs datagramas
          • IP universal, único formato de paquete
          • Proveer un camino de extremo a extremo
          • Atravesar enlaces entre cada par de nodos
          • Diferentes tecnologías y formatos de mensajes
      • PDUs
        • Frames, marcos o tramas
        • Transportan datagramas
          (Normalmente, paquetes
          de nivel de red)
      • Framing
        • Acción de codificar las tramas
          de manera que el receptor pueda
          identificarlas y extraerlas del tren de bits
          que recibe de la capa física
      • Tratamiento de errores
        • Capa física afectada por problemas
          Bits creados, perdidos o cambian su valor
        • Orígenes de los errores
          • Interferencia electromagnética
          • Ruido térmico
          • Distorsión de atenuación
        • Códigos de detección de errores
          Usan redundancia
          • Paridad
            • Apto para velocidades bajas
            • Bit de paridad: errores simples
            • Doble bit de paridad: código correctivo
          • CRC
            • Protege contra ráfagas
              Hasta n-1 bits erróneos
      • Clases de enlaces
        • Punto a punto
        • Acceso múltiple
      • Normalización
        • IEEE 802.X
    • Control de Acceso al Medio (MAC)
      • Métodos de asignación estática

      • Métodos estocásticos
        • Acceso concurrente con

          • Métodos pesimistas
            • Evitan colisiones

          • ALOHA Puro
          • ALOHA Ranurado
            Más eficiente
        • Carrier Sense Multiple Access (CSMA)
          • CSMA/CD
            Ethernet "compartida"
          • CSMA/CA
      • Algoritmos MAC determinísticos
        • Polling
          Con coordinador
        • Con paso de token
          • Token bus 802.4
          • Token Ring 802.5
    • Direccionamiento

        • Direccionamiento plano

          AA:BB:CC:DD:EE:FF
      • Protocolo ARP
        • Traducir direcciones IP a MAC
          Cache ARP
          Interviene en la entrega dentro de la red
    • Tecnologías de enlace
      • Ethernet
        Medio de broadcast
        No confiable
        IEEE 802.3
        • Ethernet "compartida"
          • Topología física de bus y luego de estrella
            Coaxil, hubs
            CSMA/CD
            Ancho de banda compartido
            Velocidades típicamente 10Mbps
        • Bridges
          • Particionan dominios de colisión
            Llenan sus tablas de direcciones por cada interfaz
            mediante aprendizaje
            Relacionan diferentes ámbitos de enlace
            con diferentes formatos de frame y reglas MAC
        • Switches
          • Esencialmente bridges multivía
            Configuran un único dominio de broadcast
            Enlaces host-switch full-duplex
            Conservan el ancho de banda
        • Ethernet "conmutada"
          • Evolución de Ethernet compartida
            Topología de estrella con switch
            Segmentación completa
            No aplica CSMA/CD
            Ancho de banda conservado
            Velocidades 100Mbps a 10Gbps o más
        • STP, Spanning Tree Protocol (802.1d)
          • Tolerancia a fallos usando redundancia
            Elimina ciclos del grafo de conectividad desactivando enlaces
            Algoritmo de árbol de cubrimiento
            Reconstruye automáticamente la conectividad ante caídas de enlaces
        • Redes virtuales (VLANs)
          • Una VLAN = un dominio de broadcast
            Una VLAN = un espacio de subred IP
            Selección de nodos por MAC, por port, por IP
            Necesitan routers para comunicarse
            Muchas VLANs pueden compartir un cableado
            Independiza la red IP del tendido físico
      • Redes inalámbricas 802.11
        • Arquitectura
          • BSS
            • Basic Service Set, conjunto de estaciones y AP
          • AP
            • Access Point
          • DS
            • Sistema de Distribución
              Conecta APs y permite roaming (movilidad)
          • STA
            • Estaciones
          • IBSS
            • Conjunto independiente
        • Modos
          • Infrastructure
          • Ad Hoc
        • Asociación
          • AP publica SSID
          • STA y AP intercambian Probe Request/Response
          • Se repite con cada AP durante el roaming
        • 802.1X
          • Supplicant envía Authentication Request
            Authenticator responde
        • Tramas
          • Hasta 2324 bytes
            Típicamente 1500 para inyectar en DS Ethernet cableado
          • Tipo de tramas
            • Management
              • Beacon o baliza
              • Association Request/Response
              • RTS/CTS (CSMA/CA)
            • Datos
              • Frame Control
                Duration
                Tres direcciones
                Payload (carga útil)
